# South Florida Tech Hub landing page

A bespoke one-page pitch built to feel native to techhubsouthflorida.org. It uses the Tech Hub's own Barlow Condensed and Open Sans typefaces, the four-color palette (magenta, cyan, yellow, dark green) pulled from their live Divi CSS, hard-edged offset shadows, italic green headings, and a subtle palm-leaf corner motif. The copy is section 7 of the fit analysis reflowed for web, opening with the verbatim headline "A note for the Tech Hub from a regional peer" and closing with a Calendly CTA for Matt Creamer, CRO. Eduba branding appears only in the footer.
